Section 1.1: Downloading external Library (Pygame)

Windows
Open Command Prompt
python --version
pip install pygame
python -c "import pygame; print(pygame.ver)"

Linux (Ubuntu/Debian)
Open a terminal
sudo apt update
sudo apt install -y python3-pip
python3 -m pip install pygame
python3 -c "import pygame; print(pygame.ver)"

macOS
Open Terminal
python3 --version
brew install python
python3 -m pip install pygame
python3 -c "import pygame; print(pygame.ver)"
